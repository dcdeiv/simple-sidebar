Simple Sidebar

Ajax

WARNING

This demo-page is built to show that if sidebar has ajax elements (or dynamically added) it will works.

This feature is available only from the simple-sidebar v1.0.3

Run bower update from your workspace if you already have this component installed. Otherwise see Bower.io to install Bower and then run bower install simple-sidebar in your workspace.

Download

ZIP

GitHub:
git clone https://github.com/dcdeiv/simple-sidebar

Bower:
